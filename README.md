Ne Bu? / What Is This?
Bu proje, HTTP protokolünü ne kadar iyi bildiğini test eden bir escape room oyunudur. Her oda farklı bir HTTP kavramını (header'lar, encoding, method'lar, kimlik doğrulama vb.) kullanarak kilitlenimiştir. Cevabı bulmak için response body'ye değil, her yere bakman gerekir.
This project is an escape room game that tests how well you know the HTTP protocol. Each room is locked using a different HTTP concept (headers, encoding, methods, authentication, etc.). To find the answer, you need to look everywhere — not just the response body.
