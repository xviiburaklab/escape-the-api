# Escape Terminal (Frontend)

Next.js frontend for the Terminal Escape Game.
A fully retro CRT visualization for an immersive hacker experience.

## Theme & Look
- **Background**: `#081101`
- **Text**: Amber `#fffefd`
- **Effects**: Scanline overlay, screen flicker, text glow.
- **Font**: VT323

![UI Placeholder](docs/screenshot.png)

## Development

```bash
npm install
npm run dev
```

Remember to rename `.env.local.example` to `.env.local` and set the backend API URL.
